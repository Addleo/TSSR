# Module 7: Gestion des espaces de stockage

## Partitionnement

-   **Outils**:
    -   `fdisk`, `parted`.
-   **LVM**:
    -   Créer et gérer des volumes logiques.
-   **Systèmes de fichiers**:

-   **Commandes**:
    -   `mount`, `umount`.
-   **Montage automatique**:
    -   Fichier `/etc/fstab`.
