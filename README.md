# Module 8: Gestion des utilisateurs et des groupes

-   **Commandes**:
    -   `adduser`, `deluser`, `usermod`.
-   **Changement d’identité**:

    -   `chmod`, `chown`.
-   **UMASK**:
    -   Définir les permissions par défaut.
